# 🔍 Exploring Deep In Data

> A collection of Exploratory Data Analysis (EDA) projects on real-world datasets to develop a deeper understanding of data, uncover meaningful insights, and strengthen analytical thinking.

---

## 📖 About

Real-world datasets are often messy, large, and full of hidden patterns. This repository is my learning journey where I explore datasets from different domains and perform in-depth Exploratory Data Analysis (EDA).

Instead of focusing only on building machine learning models, this repository emphasizes understanding the data first—because better insights lead to better decisions and better models.

Each dataset is explored independently with the goal of answering questions such as:

- What story does the data tell?
- What quality issues exist in the dataset?
- Which features are most informative?
- Are there hidden trends or unusual patterns?
- What business or real-world insights can be extracted?

---

## 🎯 Objectives

- Develop strong data exploration skills.
- Practice working with large real-world datasets.
- Improve data cleaning and preprocessing techniques.
- Build intuition before applying machine learning.
- Document findings in a structured and reproducible manner.

# 📂 Dataset Explorations

## 1. ChatGPT Reviews Dataset

**Dataset Overview**

- **Rows:** 1,048,575
- **Columns:** 8
- **Domain:** User Reviews
- **Description:** Reviews collected during the early release phase of ChatGPT, providing insights into user opinions, ratings, application versions, and usage patterns.

### Exploration Focus

- Dataset structure analysis
- Missing value analysis
- Data quality assessment
- Distribution of ratings
- Review length analysis
- User engagement patterns
- Version-wise trends
- Time-based analysis
- Correlation between features
- Business and product insights
